#### 安装教程

环境 python 3.6

#### 使用说明
1. 建议将以及需要输入的东西保存为字符串
2. 自己电脑获取ip只需将getip.py丢到电脑启动项（百度）
3. 开机 自动联网 获取ip 指定邮箱 走人
4. 若是提醒别人的请选择bat命令
